Defining color values

colors in css most often specifies in the following formates

  1. a COLOR
  2. a HEXA value -like- "#000","#fff","#c5c5c5",ect
  3. a RGB values ( RED, GREEN,BLUE)- like "rgb(0,0,0,1)", "rgb(181, 0, 45)","rgb(59, 0, 104)", ect
