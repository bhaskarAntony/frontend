javascript Ecapsulation

It is process of binding the date with the functions acting on that data. it allows uds to controll the data and validate it.

Points to remember
  • Use var keyword to make data member private.
  • use setter method to set the data and getter method to get that data.

Encapsulation allows us to handle an object using the following properties.

Read/Write - Here, We use setter method to use the data and getter methods read that data.

Read Only - in this case , we use getter methods only.

Write Only - in this case, we use setter methods only
